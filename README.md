# 🎤 VOCAL — 나만의 노래연습장

브라우저에서 바로 실행되는 개인 노래연습 프로그램. `index.html` 하나로 동작합니다 (설치·인터넷 불필요).

## 실행

`index.html`을 더블클릭하면 끝. 모든 데이터(곡, 음원, 키 설정)는 브라우저에 저장됩니다.

## 기능

- **가사 + 기타코드 리드시트** — 가사에 `[G]`처럼 코드를 넣으면 가사 위에 코드 표시
- **반음 키 조절 (♯/♭)** — 노래방처럼 반음씩 키를 올리고 내림. 소리와 코드가 함께 이조됨
- **남녀 키 전환** — 여→남(−5), 남→여(+5) 원터치 전환
- **MP3 재생** — 곡마다 음원 파일 연결, 키·템포를 바꿔도 음원에 실시간 적용
- **템포 조절** — 키 변화 없이 50–150% 속도 조절
- **노래방식 가사 싱크** — [⏱ 싱크 찍기]로 재생하며 각 줄 시작 순간 Space만 누르면 타이밍이 자동 기록됨. 이후 재생 시 현재 줄이 강조되고 다음 줄까지 색이 차오르는 노래방 스타일 표시. ±0.1초 미세조정 가능
- **싱크 가사 자동 검색** — 편집 창에서 제목·가수 입력 후 [🌐 싱크가사 자동 검색]을 누르면 LRCLIB 공개 데이터베이스에서 시간표가 붙은 가사를 통째로 가져옴. .lrc 파일 가져오기도 지원

## 가사 작성법

```
# 후렴                     ← #으로 시작하면 구간 제목
[G]아리랑 아리[C]랑 아라[G]리요   ← 코드는 붙는 위치에 [코드]
@0:32 [G]나를 버리고...          ← 줄 앞 @분:초 = 시간태그(선택)
```

## 단축키

Space 재생/일시정지 · ↑/↓ 키 올림/내림 · ←/→ 5초 이동

## 기술

단일 HTML + Vanilla JS. 피치시프트는 [SoundTouchJS](https://github.com/cutterbl/SoundTouchJS) (LGPL-2.1) 내장. 곡 데이터는 IndexedDB에 저장.
